## 使用方法

1. 从URL下载配置文件

   <https://raw.githubusercontent.com/ycg31/conf/master/Loon/Main.conf>

2. 继续编辑，将配置文本`[Remote Proxy]`中的服务器替换为您的订阅链

3. 打开Loon的MITM开关 - 证书管理 - 安装CA证书；然后到系统设置 - 已下载描述文件 - 安装

4. 最后系统设置 - 通用 - 关于本机 - 证书信任设置，信任刚才安装的证书
